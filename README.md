# 🚀 Personal Portfolio & Skills API 

Ye ek **Live Backend API** hai jise maine FastAPI aur MongoDB ka use karke banaya hai. Is project ka main goal meri technical skills ko cloud par manage karna aur automation workflows ke liye ek base taiyar karna hai.

## 🌟 Live Demo
Aap meri API ko yahan check kar sakte hain:
👉 [Live API Link](https://my-first-api-production-42c4.up.railway.app/my-info)  
👉 [Interactive Docs (Swagger)](https://my-first-api-production-42c4.up.railway.app/docs)

## 🛠 Tech Stack
- **Language:** Python 🐍
- **Framework:** FastAPI
- **Database:** MongoDB (Cloud)
- **Deployment:** Railway Cloud 🚄

## 📌 Key Features
- **Live Deployment:** Poori tarah internet par active hai.
- **Data Integration:** Cloud-hosted MongoDB ke saath connected hai.
- **Automation Ready:** Ise aage chal kar **Gobble Cube** project mein use kiya jayega.

## 🛤 API Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/my-info` | Meri professional details (Role, Goals) dikhata hai. |
| POST | `/add-skill` | Database mein naya skill add karta hai. |
| GET | `/get-skill` | Saare added skills ki list fetch karta hai. |
| GET | `/docs` | API documentation interface. |
